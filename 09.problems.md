## Problems with Terraform

| Problem | Details |
|---------|---------|
| State file management | Difficult to manage, especially in teams |
| Manual changes not detected | Console changes won't be auto-corrected by Terraform |
| Not GitOps friendly | Manual resource changes won't reflect in GitHub |
| Large projects are hard to manage | Difficult to debug |
| No rollback | If it fails halfway, half the resources are already created |
